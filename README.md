# aap-kalkulator
Kort beskrivelse av hva prosjektet dreier seg om.

# Komme i gang

Hvordan bygge, teste og kjøre koden viss aktuelt.

```bash
npm run dev
# or
yarn dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
